My e-Trading Adapters Library

For decades the Financial Industry has been reinventing the wheel over and over. Everybody has been coding the same connectors to the same exchange interfaces with no added value whatsoever at tremendous costs. MeTAL goal is to end this nonsense and start sharing these adapters.
The project DNA revolves around 3 Pillars: Open Source, High Performance and Community Driven.

Free

MeTAL is Open Source, available under the GNU GPL v2 licence. In other words, you may use, modify and distribute it. You may not sublicense or hold anybody liable. Yes you got that right. Free as in beer! Source code is available on GitHub.

  • View the GitHub Project

FIX like

MeTAL is inspired by FIX message flow and terminology, but is not tied to the protocol. All our adapters speak exchange native format. Prior knowledge of FIX will be an asset to manipulate business messages and fields.

  • QuickStart

Fast

MeTAL has been built for performance and ease of use. Send your first native protocol order in few minutes while messages are encoded in below one microsecond.
